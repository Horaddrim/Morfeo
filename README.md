# Morfeo

A simple CLI to analyze the guts of your application!

### How do I get set up? ###

* Golang >= 1.9
* UNIX based OS
* No Tests yet :camel:
* For now there are only two commands the command *build* and *help*

#### COMMANDS FOR RUN THE PROJECT ####

* `git clone https://github.com/Horaddrim/morfeo` 
* `cd morfeo`
* `go build .`
* `./morfeo build <args>`
* Enjoy! :smile:
